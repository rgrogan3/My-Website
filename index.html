Around the Web

Bill-ger Gameboy game

I created this game using C for my CS 2110 class this past summer. I was given a build file and emulator to use, and the assignment was open ended so that I could create any game fitting the requirements. I created a frogger style game with 4 rows of car/bus obstacles, 4 rows of lilypad/log obstacles and 5 homes to fill to win the game. The zip file containing all game code is here in a .gz, and can be compiled/played using the Visual Boy Advance emulator or others.

  • Reason Behind Creation: CS 2110 class
  • Date: July 2016

Master of the Chords Web App

I developed this website at Mhacks with the University of Michigan, working with a team of 4 and a time constraint of 36 hours. The website can be found here, and is designed to be used by musicians. They can create songs, or view and play songs that others have created. My work on this project was primarily with the front end: I created the Home page, About page, Contact page and Profile page. My work was done in CSS, HTML, and a little bit of PHP using Bootstrap and Laravel. I also got exposure to the back-end frameworks and languages used including firebase and jquery.

  • Reason Behind Creation: Mhacks - Refactor
  • Date: February 2016

Chess program

I developed the GUI for thi chess program for my class CS 1331. The logic for piece movement was given to me and the pieces were pre-defined. From there I added the board itself, the physical movement of the pieces, the move options, gamestate information, pawn elevation at the end of the board, page aesthetics, etc. You can download the zip file here. Run information is in the text file 'Index'.

  • Reason Behind Creation: CS1331 class
  • Date: November 2015
